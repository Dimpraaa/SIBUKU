# SIBUKU

**Sistem Informasi Buku Perpustakaan (SIBUKU)** adalah aplikasi mobile berbasis Android yang dirancang khusus untuk mempermudah manajemen data buku di perpustakaan. Aplikasi ini dikembangkan sebagai pemenuhan Tugas / Ujian Akhir Semester (UAS) mata kuliah Pemrograman Mobile.

## Fitur Utama
- **Manajemen Data Buku:** Tambah, edit, hapus, dan lihat data buku perpustakaan dengan mudah.
- **Pencarian Cerdas & Real-time:** Cari buku berdasarkan Kode Buku, Judul, atau Nama Penulis secara instan (*real-time search*).
- **Auto-Generate Kode Buku:** Algoritma pintar untuk menghasilkan urutan kode buku (contoh: BK001, BK002) secara otomatis yang mampu merapikan dan mengisi celah urutan yang terhapus (*fill the gap*).
- **Antarmuka Premium & Modern:** Desain UI/UX yang cantik, intuitif, dan sudah sepenuhnya mendukung fitur layar *Edge-to-Edge* pada perangkat Android terbaru.
- **Penyimpanan Lokal yang Aman:** Menggunakan arsitektur *database* SQLite bawaan Android sehingga aplikasi sangat ringan dan dapat berjalan secara *offline* 100%.

## Teknologi yang Digunakan
- **Bahasa Pemrograman:** Kotlin
- **IDE:** Android Studio
- **Database:** SQLite (dengan kustomisasi `SQLiteOpenHelper`)
- **Desain UI:** XML Layouts & Material Design Components

## Cara Menjalankan Proyek
1. *Clone* repositori ini ke komputer Anda:
   ```bash
   git clone https://github.com/Dimpraaa/SIBUKU.git
   ```
2. Buka proyek tersebut menggunakan **Android Studio**.
3. Tunggu hingga proses sinkronisasi *Gradle* selesai dengan sempurna.
4. Klik tombol **Run** untuk menjalankan aplikasi di Emulator atau perangkat Android fisik Anda.
